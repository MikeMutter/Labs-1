1) Using Python and s3270 write a script to logon to the mainframe

	After Logging on issue the ‘LISTCAT’ command

a) Using Python:

	import tn3270lib
	tn = tn3270lib.TN3270()
	tn.initiate('<IP address>',<port>)
	tn.send_cursor('TSO')
	tn.send_cursor('USERNAME')
	tn.send_cursor('PASSWORD')
	tn.get_all_data()
	tn.print_screen()
	tn.send_cursor('LISTCAT')
	tn.get_all_data()
	tn.print_screen()

b) Using s3270:

        Note: Notice the use of Y: here instead of L: to connect. What does the 'Y:' do?
        Also Note: If this doesn't work on 992 what other port would this work on?

	#!/bin/bash
	# $1 = tn3270 Y:<host>:<port>
	# $2 = Userid
	# $3 = Password

	s3270 $1 << EOF
	Wait(InputField)
	String(tso\n)
	String($2\n)
	PrintText(string)
	String($3\n)
	Wait(InputField)
	PrintText(string)
	String(LISTCAT)
	Enter
	PrintText(string)
	EOF
